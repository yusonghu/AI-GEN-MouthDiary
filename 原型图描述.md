# 小鼠日记本 - 原型图描述文档

## 整体布局规范

### 页面框架
```
┌─────────────────────────────────────────────────────────────┐
│  [Logo: 小鼠日记本]  小鼠管理 | 实验记录 | 数据统计 | 导出数据  │ ← 导航栏 (60px高)
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    [页面主体内容区域]                        │ ← 白色背景，最大宽度1200px
│                      (内边距: 20px)                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 颜色应用
- 导航栏：白色背景 #FFFFFF，底部边框 #E0E0E0
- 页面背景：浅灰 #F5F5F5
- 主按钮：蓝色 #2196F3，白色文字
- 次按钮：白色背景，蓝色边框
- 危险操作：红色 #F44336
- 成功提示：绿色 #4CAF50
